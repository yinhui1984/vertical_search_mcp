## 🎯 核心要点

### 项目定位
作为"双引擎知识加速系统"的**引擎 2**（前沿信息追踪），提供：
- 垂直平台内容搜索能力
- 高扩展性架构
- 生产级性能和稳定性

### 默认支持平台
- ✅ 搜狗微信公众号 (weixin.sogou.com)
- ✅ 搜狗知乎 (zhihu.sogou.com)

### 核心改进
1. **性能优化**: 浏览器复用，速度提升 5 倍
2. **插件化架构**: 新平台接入 1-2 小时
3. **配置化选择器**: 无需修改代码
4. **缓存机制**: 5 分钟缓存，重复查询秒级响应

## 🔑 关键决策

### 为什么用 Playwright？
- ✅ 完整浏览器指纹，反爬检测率低
- ✅ 自动处理 Cookie、JS、重定向
- ✅ 可视化调试
- ❌ 纯 HTTP 会被 403 拒绝

### 为什么只做搜狗两个平台？
- 搜狗微信是**独家数据源**
- 搜狗知乎验证**架构扩展性**
- 小红书、B站等反爬强或价值不匹配
- 聚焦最有价值的平台

## 🚀 使用示例

### Claude 调用
```
搜索微信公众号关于 "Web3 区块链" 的最新一周文章
```

### MCP 工具
```json
{
  "tool": "search_vertical",
  "params": {
    "platform": "weixin",
    "query": "Web3 区块链",
    "max_results": 10
  }
}
```

### 返回格式
```json
{
  "platform": "weixin",
  "total": 10,
  "results": [
    {
      "title": "Web3 技术深度解析",
      "url": "https://...",
      "source": "技术公众号",
      "date": "2026-01-05",
      "snippet": "..."
    }
  ],
  "response_time_ms": 1234
}
```
